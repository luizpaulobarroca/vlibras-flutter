**Step 1 — lib/src/vlibras_controller.dart modifications:**

Replace the static `_defaultPlatform()` method with a conditional import and a factory function call.

Add at the top of the file (after existing imports):
```dart
import 'platform/unsupported_platform.dart'
    if (dart.library.js_interop) 'platform/web_platform.dart';
```

Replace the `_defaultPlatform()` static method and constructor so the controller wires its own status callback into the platform:
```dart
VLibrasController({VLibrasPlatform? platform})
    : _platform = platform ?? createDefaultPlatform(_defaultStatusCallback);

// Placeholder — real callback assigned after _platform is set.
// Actual onStatus callback forwards to _setValue.
static void _defaultStatusCallback(VLibrasStatus _) {}
```

Actually, the correct approach is: the controller needs to pass its own `_onPlatformStatus` callback to the platform. But with the conditional import factory approach, the callback is passed at construction time. The problem is that `_onPlatformStatus` is an instance method, not callable as a static function before the instance is created.

Use an initializer that captures `this`:
```dart
VLibrasController({VLibrasPlatform? platform}) {
  _platform = platform ?? createDefaultPlatform(_onPlatformStatus);
}
```

And change `final VLibrasPlatform _platform;` to `late final VLibrasPlatform _platform;`.

Add the `_onPlatformStatus` method:
```dart
void _onPlatformStatus(VLibrasStatus status) {
  _setValue(_value.copyWith(status: status));
}
```

Add `attachElement` method (web-only; forward to platform cast):
```dart
/// Called by [VLibrasView] to attach the DOM element to the platform.
///
/// Only called on Flutter Web. The [element] is the HTMLDivElement created
/// by HtmlElementView.fromTagName. Delegating to the platform allows
/// VLibrasWebPlatform to call player.load(element).
void attachElement(Object element) {
  // Cast is safe — this method is only called from VLibrasView on web.
  (_platform as dynamic).attachToElement(element);
}
```

Note: Using `as dynamic` avoids importing web-only types in vlibras_controller.dart. The cast is only reached on web where VLibrasWebPlatform is the concrete type. This is an intentional discretion choice (structural typing via dynamic dispatch).

**Step 2 — lib/src/vlibras_view.dart:** Create VLibrasView StatefulWidget:

```dart
import 'package:flutter/widgets.dart';
import 'package:web/web.dart' as web;
import 'vlibras_controller.dart';

/// A Flutter Web widget that renders the VLibras 3D avatar.
///
/// Fills the constraints provided by its parent. Use [SizedBox], [Expanded],
/// or any sizing widget to control dimensions.
///
/// During [VLibrasStatus.initializing], the area appears blank or black.
/// Use [ValueListenableBuilder] with [VLibrasController.value] to show a
/// loading indicator outside this widget.
///
/// Example:
/// ```dart
/// SizedBox(
///   width: 400,
///   height: 300,
///   child: VLibrasView(controller: controller),
/// )
/// ```
class VLibrasView extends StatefulWidget {
  const VLibrasView({super.key, required this.controller});

  final VLibrasController controller;

  @override
  State<VLibrasView> createState() => _VLibrasViewState();
}

class _VLibrasViewState extends State<VLibrasView> {
  void _onElementCreated(Object element) {
    final div = element as web.HTMLDivElement;
    div.id = 'vlibras-player';
    div.style.width = '100%';
    div.style.height = '100%';
    div.style.background = 'transparent';
    widget.controller.attachElement(div);
  }

  @override
  Widget build(BuildContext context) {
    return HtmlElementView.fromTagName(
      key: const Key('vlibras-player-view'),
      tagName: 'div',
      onElementCreated: _onElementCreated,
    );
  }
}
```

`background: transparent` is a best-effort transparency attempt. If Unity WebGL doesn't support it, the fallback is black (documented limitation per CONTEXT.md).

library;

import 'package:flutter_test/flutter_test.dart';
import 'package:vlibras_flutter/vlibras_flutter.dart';

import 'mocks/mock_vlibras_platform.dart';

void main() {
  testWidgets('VLibrasView has Key vlibras-player-view', (tester) async {
    final platform = MockVLibrasPlatform();
    when(() => platform.initialize()).thenAnswer((_) async {});
    final controller = VLibrasController(platform: platform);

    await tester.pumpWidget(VLibrasView(controller: controller));

    expect(find.byKey(const Key('vlibras-player-view')), findsOneWidget);

    controller.dispose();
  });
}
```
